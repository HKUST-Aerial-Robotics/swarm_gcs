# Build
Prerequirements

```bash
pip install tornado==4.5.3 --ignore-installed
npm install
```

Build

```bash
build.sh
```

# Usage

Download executable from Release and unzip it


```bash
cd swarm_gcs-linux-x64
roslaunch launch/swarm_simulation.launch
```

```
cd swarm_gcs-linux-x64
./swarm_gcs
```
